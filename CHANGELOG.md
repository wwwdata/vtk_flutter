## 0.1.0-dev.1

- Add typed Dart sessions for oblique MPR, 3D volume, and volume-locator
  rendering.
- Add a monolithic VTK 9.5.2 code asset selected and checksum-verified by a
  Dart build hook.

  vtk.js web implementation.
- Add Swift Package Manager and CocoaPods support for both Apple platforms.
- Add a synthetic renderer-lab example, native contract tests, consumer build
  coverage, and immutable native-release automation.
